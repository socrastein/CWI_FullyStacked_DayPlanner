# CWI_FullyStacked_DayPlanner
A team project for CWI's CPSC-208 Intro to Full Stack Web Development course. 

Bryan, Allan, Cody, Bridger, Caleb, and Matt will all work together to create
a simple day planner that shows a weather forecast. 

[Check out the live page](https://socrastein.github.io/CWI_FullyStacked_DayPlanner/)

<br>

Inititial plans are to implement the following features: 

<br>

### Core features
- header with day of week, date, and month
- create, edit, and delete events & reminders
- view of the day with events attached to time slots
- visual indication of where current time is in the day
- basic weather forecast for the day (temp, precipitation)
- Lucide icons for UX  

<br>
  
### Extra features
- categories and color-coding for events
- change view to show the week or the month
- news ticker with top 1-3 headlines of the day
- drag events to move to different time slot
- swipe planner to show previous/next day/week/month
- subtle animations to enhance UX
- select color theme, light/dark mode
